Zones

Zones are provided for two main reasons:
  1. To allow you to track statistics for different sized banners across all of your web pages, and
  2. To show multiple ads in DIFFERENT areas(zones) on the same web page. 

When placing multiple ads on the same web page, you must use a unique zone for each area that will be showing an ad.  If you don't you might see the same ad appearing twice.  Below is a sample web page showing one of many possible layouts.  Note that each zone is distinct, beginning with zone1 at the top and ending with zone 4.  Actual zone numbers will vary depending on how many you have added to your web site.

Zone Information    
Indicates Required Fields.
Zone Name: Enter a name for this zone.  For example, if the zone is your home page, you might use "Home Page". 
Option Description/ URL: Enter an optional description or Page URL for information purposes only.
Mode: Enter the mode to be used on this page.   SSI is short for Server Side Include mode.  By default, Internet Information Server supports Server Side Includes in ASP files.  However, the default does not support ASP Server Side Includes in HTML files.  In this case you will want to serve ads using the advanced HTML ad serving code.
Zone Width: Enter the Width of the zone.  For example, if this zone is showing 468 X 60 banner ads, then enter the width as 468.  This information if vital for Ban Man Pro in order to choose the proper default campaign if no valid banner ads are found.
Zone Height: Enter the Height of the zone.  For example, if this zone is showing 468 X 60 banner ads, then enter the height as 60.
Campaign(s): Select all evenly distributed campaigns to include in this zone.  To select multiple campaigns, simply hold the CTRL key while clicking on each campaign.  Evenly distributed campaigns take priority, followed by weighted campaigns.  Ban Man Pro first attempts to find if there are any even campaigns that must be served at any given moment.  If no valid even campaigns are found, then Ban Man Pro next attempts to find a valid weighted campaign for this zone.  It is important that you do not oversell your ad space otherwise some evenly distributed campaigns will not deliver in the allocated time.

Click for more information on the smoothing algorithm.

Click for information on how Ban Man Pro selects campaigns.

Campaign(s): Select all Weighted campaigns to be included in this zone.  Enter the weighting of each campaign within this zone.  Valid values are any integer between 1-100.  For example, suppose you have two advertisers running advertising campaigns in a given month.  You might choose to show one 60% of the time and the other 40% of the time.  In this case you would enter values of 60 and 40 respectively.  The sum of the weightings for all selected campaigns cannot exceed 100.  Also, the sum of all weightings should be 100.  If less than 100, the default will be selected.  If no default is included, a blank 1X1 pixel image will be served.

Note:  Should you desire to create an empty zone for purposes of tracking the number of hits on a page, simply create a zone with no campaigns selected.

Include Default Campaigns: Check this option to include the default campaign(s) in this zone.  A default is used in the event that a valid campaign is not found, or in the even that your weightings sum to less than 100.

Ad Serving Code
Zone Code (Where Do I Find the Code to place on my Web Pages)
The Ban Man Pro ad serving code can be obtained by doing the following:
  1. Select the Zones folder.  A list of all zones will be shown.  If they are not, click on View All to view all zones.
  2. On the left side of each zone, you will find a blue button labeled "CODE".  Click on that button to obtain the code for this zone.
  3. Now, copy and paste the code to the web page on which you want to show the ads for this zone.
