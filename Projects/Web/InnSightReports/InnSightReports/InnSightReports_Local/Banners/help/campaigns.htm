Campaign Information
Indicates Required Field

Campaigns simply group banners for a single advertiser into a "Campaign" which defines the flight dates, distribution, quantity, cost, etc.  A campaign also defines the weightings of banners should an advertiser have more than one banner.  Campaigns are specific to an advertiser, yet a single advertiser could have multiple campaigns.
Advertiser: Select the advertiser this campaign belongs to.
Campaign Type Select the type of campaign.  The default type is Banners but campaigns can also be of type Static Text. 
Campaign Name: Specify a descriptive name for this campaign.  If an advertiser will only have one campaign you might choose the Advertiser Name.  Ban Man Pro automatically defaults to the advertisers name.  Note that campaign names must be uniqe.
Banners: Select all banners to be rotated in this campaign.  Enter the weighting for this campaign.  Valid values are INTEGERS between 1-100.  The sum of all banner weightings cannot exceed 100 and SHOULD sum to 100.  When the sum is less than 100, there will be instances where no banners are shown for this campaign.  For example, if I select two banners each weighted as 20 percent.  The sum is 40 percent indicating that 60 percent of the time no banner would be shown for this campaign.  If this occurs, Ban Man Pro attempts to use any default campaigns which you have specified. 

NOTE:  All banners within a campaign must be the same size.  The size will be listed in parentheses next to the banner name.
Default Campaign: Check this box if the campaign is being setup as a default.  A default campaign is used if no valid campaigns are found at any given moment. For example, if I am running 5 campaigns, and all have expired for one reason or another, no banner would be shown in SSI mode, and a broken link would be shown in HTML mode. To prevent nothing from being shown, or a broken link, you could setup a default that would be shown. Typically this might be something advertising something internal to your company. It is a very good idea to setup a default campaign and include it in every zone to prevent broken links or void space on your web site.

Note:  It is recommended that you have 1 default for each banner size running on your site.  This will prevent the occurrence of white space on your web pages.
Start Date: Enter the start date for the campaign.  Note:  You can schedule campaigns to begin in the future.
End Date: Enter the end date for the campaign.  The End date should be later then today's date otherwise the campaign is considered expired.
Daily Start/End Time: Select the period of time to display the ad.   For example to display a campaign between 9:00 AM and 5:00 PM, select the appropriate times from the drop-down list.  ** Setting the start time equal to 12:00 AM and the end time equal to 12:00 AM will cause the banner to rotate during all hours.
Days Selected: Select the days to show the campaign.   For example, the advertiser might choose to display an ad only on weekdays.   In this case, select Monday, Tuesday, Wednesday, Thursday, and Friday. ** If using this option, be sure to include a default campaign in the zone that uses this campaign.  This will prevent your web pages from displaying white space if no campaigns are available.
Campaign Distribution: Select the campaign distribution type.  Choices include:
  • Evenly Distributed -- Select this distribution of you want Ban Man Pro to evenly spread the impressions over the campaign Flight Dates.  This option can only be applied to CPM type ads.
  • Weighted in Each Zone -- Select this option if you will be managing the impression of each campaign via a 1-100 weighting system.
  • Called By Keyword Only -- Select this option if the campaign is only to be called by keywords.

Click for more information on the Ban Man Pro Even Distribution smoothing algorithm.

Type: Select the campaign type. 
  • CPM -- Cost Per Thousand campaign.   Select this campaign when the advertiser is paying you based on the number of impressions.  For example, if the advertiser has purchased 5000 banner impressions and is paying you $30 per thousand, select CPM.
  • Per Click -- Select Per Click if the advertiser is paying you each time a user clicks on the banner ad.  For example, if the advertiser is paying $0.50 cents each time a user clicks on the ad, then select Per Click.
  • Flat Rate -- Select Flat Rate if the advertiser is paying you a flat rate for the advertising campaign.  For example, if you sold the advertiser all impressions on your homepage for $1000, then select Flat Rate.
Quantity: Enter the quantity sold to the advertiser.   For CPM type campaigns, enter the number of impressions sold.  For Per Click campaigns, enter the maximum number of clicks the advertiser has committed to.  For Flat Rate campaigns, there is no need to enter a value as campaign validation is based solely on the Start/End Date and Times as well as the days selected.
Cost: Enter the cost to the advertiser.  For CPM type campaigns, enter the cost per thousand.   For Per Click campaigns, enter the Cost Per Click.  For Flat Rate campaigns, enter the full amount being charged to the advertiser.
Keywords: Enter keywords for this ad if the Campaign Distribution is Call By Keyword Only.  This will allow you to call this banner advertisement based on keywords.  This option is useful if running a search engine and selling ads based on a particular search phrase.  Note that campaigns called by keyword cannot be displayed in a particular zone and can only be displayed by keyword.  Click here for information on calling a campaign by keywords.
