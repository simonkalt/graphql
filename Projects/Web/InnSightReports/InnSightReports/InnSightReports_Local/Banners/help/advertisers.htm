Advertisers

Company Name: Enter the Advertising Companies name. 
Description: Enter an optional description for the company.   This is for information purposes only and is not utilized by Ban Man Pro.
Website: The companies web site.  This is for information purposes only and is not utilized by Ban Man Pro.
Contact: Enter a contact name for the advertising company.  This is for information purposes only and is not utilized by Ban Man Pro.
Email: Enter the email address of the contact.   This will be used by Ban Man Pro for emailing reports.
Login Name: Enter the Login Name to be used by the advertiser for viewing real-time statistics for their advertising campaign.
Login Password: Enter the Login Password to be used by the advertiser for viewing real-time statistics for their advertising campaign.
Address Informatioin: Enter the companies address information.
Telephone/Fax: Enter the companies telephone and fax.

Email Daily Report: By checking this option the advertiser will receive an email summary of the previous days campaigns.  An advertiser will only be emailed campaigns which belong to them.

Here is a sample report...

Campaign Name        Impressions      Clicks      Click Rate
------------------------------------------------------------
Campaign1               52               1           1.92%
Campaign2               14               0           0.00%
------------------------------------------------------------
Totals                  66               3           0.00%

 
Email Weekly Report: By checking this option the advertiser will receive a weekly report mailed once each week.  The report is emailed on Sunday and summarizes the previous week covering Sunday-Saturday.  The format is identical to the Daily Report shown above.
