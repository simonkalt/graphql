Banners

Note:  Because the Banner records are specific to an advertiser, you must add at least one advertiser before specifying a banner.  Indicates required fields.

General Required Information

Advertiser: Select an Advertiser from the drop-down menu.
Banner Ad Description: Enter a description for the banner.  If an advertiser only has a single banner you might use the same name as the advertiser.
Target URL: Enter the target URL for when a visitor to your site clicks on this specific banner.  Be sure to enter the fully qualified URL including the http://. 

For example, if you are running an advertisement for Microsoft, the Target URL might be http://www.microsoft.com 
Type of Ad Select Image Ad if this is a standard image advertisement.  To create a static text link type ad, select  Text Link Ad.

Image URL: Enter the URL for the banner ad to be displayed.  Use the fully qualified URL rather than a virtual URL.  This is necessary if you plan on displaying a banner ad on multiple pages that reside in different directories within your web site.
Alt Text: Enter any optional text to display when the users hovers the mouse over the banner ad.
Height: Enter the height of the image in pixels.
Width: Enter the width of the image in pixels.
Border: Enter the image border (typically zero).
Alignment: Enter the banner alignment.
Optional Text Underneath: Enter any optional text to display underneath the banner ad such as "Visit Our Sponsors".  If the user clicks on the text, it will be treated as a click-thru just as though the user clicked on the corresponding image.
Launch In New Window: Select Yes if you would like to launch a new instance of the user's browser when they click on the banner ad.  This will help prevent them from leaving your site.

Link Text:  If creating a text link, specify the text that will appear as the link.
Advanced Banner Option
Advanced: The advanced option allows you to specify your own ad code.  This feature is also commonly used for third-party ad code such as BurstMedia, FlyCast, etc.  Click here for more information on this topic.
