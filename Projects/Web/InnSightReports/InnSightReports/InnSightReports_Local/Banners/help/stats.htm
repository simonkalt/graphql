Reports/Stats

The Ban Man Pro Stats/Reports page begins by presenting a summary of all Advertising Campaigns followed by the option to create more detailed reports. 
Campaign Name: Select All Campaigns or an individual campaign to report.
Report Type: Select the type of Report.  Report types include:
  • Summary By Day -- Provides a summary of impressions/clicks for the selected campaign(s) during the selected period.
  • Summary By Banner -- Provides a summary of impressions/clicks for the selected campaign(s) broken down by each banner within the campaign during the selected period.
  • Summary By Banner By Day --  Provides a summary of impressions/clicks for the selected campaign(s) broken down by each banner within the campaign and by each day during the selected period.
  • Summary By Zone -- Provides a summary of impressions/clicks for the selected campaign(s) broken down by Zone within the campaign during the selected period.
  • Summary By Zone By Day --  Provides a summary of impressions/clicks for the selected campaign(s) broken down by each Zone within the campaign and by each day during the selected period.
  • Click Detail -- Provides the details of each Click-Thru during the specified period for the selected campaign.  Information includes the Date/Time, Browser Type, and the User's IP Address.
  • Executive -- The executive report is only available to administrators and provides an overall summary of all campaigns for the specified period.  The report also provides a summary for each campaign since the campaign start date.  Lastly, a summary of all campaigns with future start dates is displayed.
  • Billing Summary --  The billing summary simply provides a cost estimate based on the campaign type and the cost for the campaign.  Unless you enter the cost of each campaign this feature is not very useful.
Format: Select HTML to view the report in your browser, or Excel to export the report to Microsoft Excel.
Start Date: Select the Start Date.  Note that this field defaults to today's date.
End Date: Select the End Date.  Note that this field defaults to today's date.
